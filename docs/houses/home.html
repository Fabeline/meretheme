## Home Welcome to MereTheme - Your simple frontend framework. ### What is MereTheme? - A frontend framework that provides everything you need to create simple websites with minimal effort - No server or npm needed. Just open the HTML file in your browser - No build process - No JavaScript knowledge required - SEO friendly - Accessible - Responsive - Open Graph support ### Features - Reuse components across pages - Reuse the top and bottom of all your pages without a page refresh - Internal linking between pages without a total page refresh - Built in integrations - Routing - Share your pages on social media ### Built in integrations These integrations will only be included if specified. Note that you are not limited to these integrations. You can include any library you want, and use whatever styles and scripts supported by HTML like JavaScript. - Markdown - for easier content creation - Bootstrap - for built in styling - FontAwesome - for icons - Prism - for highlighting code
This documentation is created with MereTheme and is included in the GitHub repository as an example.
