Sharing your pages

To be able to create a nice shareable object in social media, you can use the open graph protocol. Read more about OG. Since we use iframes for implementation we need to do some special handling around this to be able to have different OG tags for each page (house). That is why we have integrated social media buttons into the framework. Look at the bottom of this page to see a preview of the buttons. Note that you should share the link to the house file and not the index file when sharing a link to a page.

1. Add your OG tags to the <head> of the house files. Example:

<meta property="og:title" content="Home" />
<meta property="og:description" content="A simple open source frontend framework requiring no server or build time." />
<meta property="og:url" content="https://fabeline.github.io/meretheme/docs/houses/home.html" />
<meta property="og:type" content="website" />
<meta property="og:site_name" content="MereTheme" />

2. Add social media buttons somewhere in the <body> 

<div 
    data-mt-type="share" 
    data-mt-value="copy, facebook, twitter, linkedin"
></div>

Translations

To be accessible, the icons have aria-labels. If you want to change the labels (for instance translations), you can do so by adding them to the attribute "data-mt-translations" in the same order as you specified the media. 

data-mt-translations="copycat, fj√∏sbok, tvitre, linkedin"

The confirm message for the copy button can be changed by adding the following attribute: to the same div: 

data-mt-copy-confirm="Your content has been copied"

Search engine optimization

Other meta tags should also be added to the head of your house files to improve SEO. For instance: 

<meta name="description" content="A simple open source frontend framework requiring no server or build time.">
