FontAwesome

FontAwesome is a large icons library. MereTheme includes the free icons in version 6.5.2.
  1. Add this to the file where you want to activate FontAwesome: 
    <div data-mt-include="fontawesome"></div>
  2. Start using icons

Example

<div data-mt-include="font-awesome"></div>
<i class="fa-brands fa-fort-awesome"></i>
