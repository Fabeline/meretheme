Markdown integration

MereTheme uses the marked library to convert from markdown to HTML. 1. Add the **data-mt-markdown="true"** attribute to the div where you want to include markdown. 2. Start writing markdown directly in your HTML file

Example

<div data-mt-markdown="true">
    # This is a heading
    1. First list item
    2. Second list item
</div>

Tutorial

You can find a nice tutorial at markdowntutorial.com to get started with markdown.

A handy markdown cheat sheet can be found at markdownguide.org.
