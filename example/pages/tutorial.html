Tutorial

Welcome to the MereTheme tutorial! In this tutorial, we will cover the basics of using MereTheme to style your web pages.

Step 1: Include MereTheme files

To get started, include the MereTheme JavaScript and CSS files in your HTML file. You can do this by adding the following lines of code to the ` ` section: 

        <script src="../../lib/meretheme.js"></script>
        <link rel="stylesheet" href="../../lib/meretheme.css" />

Step 2: Define the MereTheme attributes

In the ` ` section of your HTML file, add the following attributes to the ` ` tag: 

        <body data-mt-id="home" data-mt-type="house">

The `data-mt-id` attribute specifies the ID of the MereTheme instance, and the `data-mt-type` attribute specifies the type of the MereTheme instance.

Step 3: Include external CSS files

If you want to include additional CSS files, you can use the `data-mt-include` attribute. For example, to include the "w3.css" file, add the following line of code:

        <div data-mt-include="w3.css"></div>

Step 4: Add content

Finally, you can add your content within the `

` element with the `data-mt-type="content"` attribute. For example:

        <div data-mt-type="content">
            <h2>Tutorial</h2>
            <p>This is a tutorial on how to use MereTheme.</p>
        </div>

Feel free to customize the content and styles to fit your needs.

That's it! You have now created a basic MereTheme page. You can explore more advanced features and customization options in the MereTheme documentation.
