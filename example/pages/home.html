## Home Welcome to MereTheme - Your simple frontend framework. ### What is MereTheme? - A framework for creating web pages with minimal effort - No server or npm needed. Just open the HTML file in your browser. - No build process - No JavaScript knowledge required - SEO friendly - Accessible ### Features - Reuse components across pages - Menu for switching between pages - Toggle dark mode - Built in integrations ### Built in integrations These integrations will only be included if needed - Markdown - Tailwind CSS - Bootstrap - FontAwesome
